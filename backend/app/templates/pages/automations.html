  2. 2 Activez l'API Gmail pour ce projet
  3. 3 Dans Identifiants → Créer des identifiants → ID client OAuth 2.0 - Google vous demandera d'abord de configurer l'écran de consentement : saisissez un nom d'app, votre email, choisissez Externe, et validez
  4. 4 De retour dans Identifiants → section Clients OAuth 2.0 → Créer un client, choisissez le type Application Web
  5. 5 Dans URI de redirection autorisés, ajoutez exactement : {{ gmail_redirect_uri }}
